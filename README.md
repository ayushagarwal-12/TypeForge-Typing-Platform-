# TypeForge

TypeForge is a modern touch typing platform designed to improve typing speed and accuracy through real-time feedback and visual guidance.

## Features

* Real-time WPM and accuracy tracking
* Finger placement guidance for each key
* Interactive keyboard visualization
* Error highlighting system
* Performance heatmap
* Practice modes and lessons

## Tech Stack

* HTML
* CSS
* JavaScript (Vanilla)

## Live Demo

https://type-forge-typing-platform.vercel.app/

## Installation

1. Clone the repository
2. Open index.html in your browser

## Purpose

This project was built to solve a real problem — learning proper touch typing with visual feedback and structured practice.
